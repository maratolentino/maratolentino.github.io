BLOOM: POS System

Bloom is a web-based, role-specific POS system designed to streamline flower shop operations by integrating secure checkouts, multi-channel payments, and real-time inventory tracking into a single relational database. The platform features a centralized dashboard that generates daily, weekly, or monthly sales analytics, handles customer loyalty rewards, and enforces permission-based security for admins and cashiers. While currently limited to online, single-branch English use, it eliminates manual recording errors to optimize overall business productivity and management.

Credits: Aila Quimio, Charlize Mojares, Khyle Recaña, Mariane Sanchez, Mara Tolentino

TINDAHAN: System Portal

This unified web-based portal accelerates local store operations by integrating official government compliance and business permit processing directly into a digital e-commerce platform. Store owners can seamlessly file registrations, track regulatory document approvals, and manage their storefront transactions and inventory within a single secure interface. By bridging public administrative requirements with private retail management, the platform eliminates bureaucratic delays and empowers entrepreneurs to scale their businesses efficiently.

CryptFall: Pygame

Built using the Pygame framework, this 2D game leverages a modular Python architecture to handle real-time event loops, dynamic sprite management, and pixel-perfect collision detection. The codebase integrates object-oriented programming to structure interactive puzzle-solving mechanics, frame-rate independent physics, and state-driven game menus. By managing custom rendering layers and background audio mixing pipelines, the system transforms creative art assets into an engaging, responsive digital environment.

Credits: Aila Quimio, Charlize Mojares, Khyle Recaña, Mariane Sanchez, Mara Tolentino

Form: SQLite Database

This database-driven form system is powered by SQLite to manage, query, and organize records efficiently through structured relational tables. The application implements complete CRUD functionalities, allowing users to seamlessly create, read, update, and delete entries with instantaneous data persistence. By leveraging SQLite's lightweight and self-contained architecture, the platform ensures rapid local storage handling and data integrity without the overhead of a complex server configuration.

See GitHub Repository
Credits: Mariane Sanchez, Mara Tolentino
